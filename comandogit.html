Guía Básica de Git — LENGUAJE PARA DESARROLLO WEB IUSH

Git es un sistema de control de versiones distribuido que te permite rastrear cambios y colaborar de forma segura. Esta guía resume los comandos esenciales con ejemplos breves.

Configuración Inicial

Configurar nombre de usuario y correo

git config --global user.name "Tu Nombre"
git config --global user.email "tuemail@example.com"

Verificar la configuración

git config --list

Crear y Administrar Repositorios

Inicializar un repositorio

git init

Clonar un repositorio existente

git clone <url-del-repositorio>

Trabajando con Archivos

Ver el estado de los archivos

git status

Añadir archivos al área de preparación (staging area)

git add <archivo>  # Añadir un archivo específico
git add .          # Añadir todos los archivos modificados

Realizar un commit

git commit -m "Mensaje del commit"

Ver el historial de commits

git log

Trabajando con Ramas

Crear una nueva rama

git branch <nombre-de-la-rama>

Listar ramas

git branch

Cambiar a otra rama

git checkout <nombre-de-la-rama>

Crear y cambiar a una nueva rama

git checkout -b <nombre-de-la-rama>

Fusionar una rama con otra

git merge <nombre-de-la-rama>

Trabajando con Repositorios Remotos

Añadir un repositorio remoto

git remote add origin <url-del-repositorio>

Ver los repositorios remotos

git remote -v

Descargar cambios del remoto

git pull origin <nombre-de-la-rama>

Subir cambios al remoto

git push origin <nombre-de-la-rama>

Deshacer Cambios

Quitar archivo del staging (mantener cambios)

git reset <archivo>

Deshacer el último commit (mantener cambios)

git reset --soft HEAD~1

Descartar cambios en el área de trabajo

git checkout -- <archivo>

Otras Utilidades

Ver diferencias

git diff

Crear una etiqueta (tag)

git tag -a v1.0 -m "Versión 1.0"

Ver todas las etiquetas

git tag

Subir etiquetas al remoto

git push origin --tags

Conclusión. Esta guía cubre los comandos básicos para empezar a usar Git. Practícalos con frecuencia y consulta la documentación oficial para temas avanzados.
